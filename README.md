# 🔺 Shape Shout! — Free Printable Party Game Kit

**One peek. One voice. Two pads.** A blind-drawing showdown where one player memorizes a secret picture, then draws it from memory *while shouting instructions* — and real-world words are strictly illegal. Say the shape, not the thing!

**▶ Play it now: [devkushgowda.github.io/shape-shout-party-kit](https://devkushgowda.github.io/shape-shout-party-kit/)**

Perfect for families, classrooms, youth groups, office team events, and game nights. Ages 8 → 108. No sign-up, no ads, nothing to buy — just print and play.

## 🎲 How it works

1. **Pair up & wall off** — each team is a *Guide* and an *Artist*, back-to-back. No peeking, ever.
2. **The 10-second peek** — the Guide studies a secret shape card, then it goes face-down.
3. **The 60-second chaos** — the Guide draws it from memory *while shouting geometric instructions*; the Artist draws only what they hear.
4. **Reveal** — line up the card, the Guide's drawing, and the Artist's drawing. Prepare to be humbled.
5. **Score & swap** — memory + precision + vibe vote − penalties. Every real-world word ("house!", "snowman!") costs a point.

## 🚀 Two ways to print

1. **Easiest:** grab the ready-made PDFs from the [`pdf/`](pdf/) folder and print at **100% scale** (Letter or A4 both work — don't "fit to page").
2. **Interactive:** open the [website](https://devkushgowda.github.io/shape-shout-party-kit/) (or `index.html` locally) and use the toolbar to print everything or any single section. Turn on **"Background graphics"** in the print dialog.

**Tips:** use cardstock for the shape cards so marker doesn't show through, and everything prints fine in black & white too.

## 🧰 You'll need

2 pads · 2 markers (no erasing — commit!) · a barrier or back-to-back chairs · a timer · a room full of judges.
